### Клонируйте вот тут

    git clone https://github.com/prElyor/svelte-app.git

### После клона вставляйте в терминал 
    npm install
### or
    yarn add

### После установки всех зависимостей вставляйте в терминал
    npm run dev
### or
    yarn run dev
